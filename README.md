# late-ui
## 自定义组件库
## npm安装
```
npm install @whl114/late-ui

```
## yarn 安装
```
yarn install @whl114/late-ui

```

## 启动
``` 
npm run dev
```
## 打包
```
yarn run build
```
## 登录注册npm请参照 https://docs.npmjs.com

## 更新版本号
```
npm version [<newversion> | major | minor | patch | premajor | preminor | prepatch | 

yarn version --newversion

```
# 发布npm
```
yarn run pub
```
